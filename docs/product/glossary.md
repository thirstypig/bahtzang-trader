# Glossary

Define every project-specific term, acronym, and persona here.

The test for inclusion: if someone returning to this project after three months would
have to read code to understand the word, it belongs here. Terms borrowed from trading
count — this project uses several in a narrower sense than the industry does.

## Terms

| Term | Definition | Notes |
|---|---|---|
| **Portfolio** | A virtual sub-account with its own budget, virtual cash, goal, risk profile, and kill switch. All portfolios share one real broker account. | Narrower than the everyday meaning — here it is the unit of execution. Every trade runs through one. |
| **Phase G** | The paper-trading → live-trading graduation track, and its gates. | See `CLAUDE.md` and the Phase G plan for the locked decisions. |
| **Virtual cash** | A portfolio's own cash balance, tracked independently of the real broker balance. | Lets several portfolios share one account without double-spending. |
